WEB PROGRAMMING

Web programming adalah proses membuat dan mengembangkan situs web menggunakan bahasa pemrograman seperti HTML, CSS, dan JavaScript. Tujuannya adalah membangun halaman web yang interaktif, responsif, dan mudah digunakan oleh pengguna. Seorang web programmer tidak hanya mengatur tampilan, tetapi juga mengatur logika, struktur data, dan koneksi dengan server agar website dapat berfungsi dengan baik.

HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat struktur dan konten halaman web, seperti teks, gambar, tautan, tabel, dan video. HTML menjadi fondasi utama dalam pembangunan sebuah website.

  • Diciptakan oleh Tim Berners-Lee pada tahun 1991 di CERN.
  • Versi pertama hanya memiliki sekitar 18 tag dasar.
  • HTML 4 dirilis pada tahun 1997 dan menjadi standar untuk web awal.
  • Pada tahun 2014, HTML5 diperkenalkan dengan dukungan video, audio, dan elemen semantik baru.

CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan gaya elemen HTML, seperti warna, ukuran font, jarak, dan tata letak. CSS memungkinkan pemisahan antara struktur dan desain halaman web.

  • Diperkenalkan oleh HÃ¥kon Wium Lie pada tahun 1994.
  • Versi pertama (CSS1) dirilis oleh W3C pada 1996.
  • CSS2 (1998) membawa fitur layout yang lebih kompleks.
  • CSS3 (2011) memperkenalkan modul seperti flexbox, grid, animation, dan transition.

TAILWIND

Tailwind CSS adalah framework CSS berbasis utilitas yang mempermudah pengembang membuat desain modern dan responsif dengan cepat. Framework ini menyediakan kelas siap pakai tanpa harus menulis CSS manual dari awal.

  • Dikembangkan oleh Adam Wathan dan dirilis pertama kali pada 2017.
  • Terinspirasi dari konsep utility-first untuk efisiensi dan fleksibilitas.
  • Menjadi populer karena meminimalkan kebutuhan menulis CSS kustom.
  • Sekarang digunakan luas dalam pengembangan front-end modern.

JAVASCRIPT

JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas dan logika pada halaman web. Dengan JavaScript, pengguna dapat berinteraksi langsung dengan elemen di halaman tanpa harus memuat ulang.

  • Diciptakan oleh Brendan Eich di perusahaan Netscape pada tahun 1995.
  • Awalnya bernama Mocha, lalu LiveScript, sebelum resmi menjadi JavaScript.
  • Pada tahun 1997, JavaScript distandardisasi oleh ECMA menjadi ECMAScript.
  • Sekarang digunakan di berbagai platform, termasuk server dengan Node.js.

Tata Cara Membuat Website Sederhana

1. Buat Folder Proyek

Langkah pertama dalam membuat website adalah menyiapkan struktur folder proyek. Buat sebuah folder baru di komputer, misalnya dengan nama website-saya. Folder ini berfungsi sebagai tempat utama untuk menyimpan seluruh file dan aset yang dibutuhkan website, seperti file index.html untuk struktur halaman, style.css untuk tampilan atau desain, serta script.js untuk menambahkan logika dan interaktivitas. Kamu juga bisa membuat subfolder tambahan seperti images untuk menyimpan gambar, atau assets untuk menyimpan font dan ikon. Dengan struktur folder yang rapi sejak awal, proses pengembangan website akan lebih teratur dan mudah dikelola.

2. Buat File HTML

Buat file index.html berisi struktur dasar website: 


<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Sederhana</title>
</head>
<body>
  <h1>Selamat Datang di Website Saya</h1>
</body>
</html>

3. Tambahkan CSS

Buat file style.css untuk memperindah tampilan, lalu hubungkan ke HTML melalui tag <link> di bagian <head>. Berikut ini contoh implementasi css-nya: 

body {
  font-family: Arial, sans-serif;
  background-color: #f8f9fa;
  text-align: center;
  color: teal;
  margin: 10px 20px;
  padding: 5px 2px;
}

4. Tambahkan JavaScript

Buat file script.js untuk menambahkan interaktivitas pada website. Setelah itu, sambungkan file tersebut ke index.html dengan menambahkan tag <script> tepat sebelum tag penutup </body>. Contohnya: 

<body>
  <h1>Selamat Datang di Website Saya</h1>

  <!-- Menghubungkan file JavaScript eksternal -->
  <script src="script.js"></script>
</body>

5. Gunakan Tailwind CSS (Opsional)

Tambahkan Tailwind CSS melalui CDN agar desain website lebih cepat dibuat dan tampil konsisten. Caranya, salin kode berikut dan tempelkan di dalam bagian <head> pada file index.html, sebelum tag penutup </head>: 

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Website Sederhana</title>

  <!-- Menghubungkan Tailwind CSS melalui CDN -->
  <script src="https://cdn.tailwindcss.com"></script>
</head>

6. Uji di Browser & Upload

Klik dua kali file index.html untuk membukanya di browser dan pastikan tampilannya sesuai. Jika sudah benar, unggah folder proyek ke platform hosting seperti GitHub Pages (via repository), Netlify (drag & drop folder), atau 000webhost (upload manual lewat file manager).
